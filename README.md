📊 Hospitality Revenue Insights
Data Cleaning with Python & Visualization in Power BI

This project showcases an end-to-end data analytics workflow — from data cleaning using Python to building an interactive Power BI dashboard for hospitality revenue insights.

It demonstrates practical skills in data preprocessing, transformation, and business intelligence reporting.

🔄 Project Workflow
🧹 1. Data Cleaning & Preparation (Python)

The raw hospitality dataset was processed using Python to ensure accuracy and usability.

Key steps:

handled missing values

removed duplicates & inconsistencies

formatted dates & categorical fields

standardized revenue & pricing metrics

prepared structured dataset for analysis

Libraries Used:

Pandas

NumPy
